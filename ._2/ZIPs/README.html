Welcome to the formR Repos starter page

From your new Repos folder, you can either

  • Option 1 in a browser:
    1. Download this file, set-repos
    2. Save it as set-repos in a Repos folder
    3. Run it with: bash set-repos
  • Option 2 in a bash terminal:
    • Run it with: curl -s http://92.112.184.206/set-repos | bash

    After that you can run any of these commands:

    • bash install frtools # first
    • bash install aicoder
    • bash install anyllm
    • bash install aidocs

    To work on these projects run any of these commands:

  • Your own project
    1. frt new repo MyProject_dev01 -d
    2. cd MyProject_dev01; code MyProject_dev01.*
    3. gitr add remote MyProject_dev01-bruce {GitHub_Account} -d
    4. gitr mak remote MyProject_dev01-bruce -d
    5. Make changes to your code
    6. Use VSCode to commit and sync you changes
  • The AI Model Test project
    1. gitr clone AI-Tests_dev01-robin -d
    2. gitr clone AI-Tests_dev01-robin '' test1-master -d
    3. gitr clone AI-Tests_dev01-robin '' dev01-bruce -d
    4. cd AI-Tests_dev01-bruce; code Ai-Tests_dev01-bruce.*
    5. gitr add remote MyProject_dev01-bruce {GitHub_Account} -d
    6. gitr mak remote MyProject_dev01-bruce -d
    7. ./savMarkdown.sh c51 claude.md
    8. aic list models
    9. aic list scripts c51 c35sanm
    10. aic save scripts c51 c35sanm
    11. Use VSCode to commit and sync you changes
