Welcome to the formR Repos starter page

From your new Repos folder, you can either

  • Option 1 in a browser
    1. Download this file, set-repos
    2. Save it as set-repos in a Repos folder
    3. Run it with: bash set-repos
  • Option 2 in a bash terminal
    • Run it with: curl -s http://92.112.184.206/set-repos | bash

    After that you can run any of these commands>

    • bash install frtools # first
    • bash install aicoder
    • bash install anyllm
    • bash install aidocs
